HTML LISTS

  1. Ordered List

    An ordered list starts with the ol tag. Each list item starts with the li tag.
    The list items will be marked with numbers by default:

    • Numbers (Default): The list items are marked with Arabic numerals (1, 2, 3...). This is the default behavior if no type attribute is specified.
    • Uppercase Letters: The list items are marked with uppercase alphabetical letters (A, B, C...).
    • Lowercase Letters: The list items are marked with lowercase alphabetical letters (a, b, c...).
    • Uppercase Roman Numerals: The list items are marked with uppercase Roman numerals (I, II, III...).
    • Lowercase Roman Numerals: The list items are marked with lowercase Roman numerals (i, ii, iii...).

  2. Unordered List

    3. In HTML, unordered lists are created using the ul tag, with each list item defined by the li tag. While the default bullet style for an unordered list is a filled circle (disc), the appearance of these bullet points can be customized.

    • disc: This is the default style, displaying a filled circular bullet.
    • circle: This displays a hollow circular bullet.
    • square: This displays a filled square bullet.
    • none: This removes the bullet points entirely, leaving only the list item text.

  3. Description List

    4. HTML also supports description lists. A description list is a list of terms, with a description of each term. The dl tag defines the description list, the dt tag defines the term (name), and the dd tag describes each term:

    • Single Term, Single Description: This is the most straightforward structure, where each term has a single, corresponding description.
    • Single Term, Multiple Descriptions: A single term can have multiple descriptions associated with it, which is useful for providing comprehensive details or various attributes for one item.
    • Multiple Terms, Single Description: In cases where several terms share the same description, this structure can be used to avoid repetition.
    Eg:
    Orange
    Orange is a Fruit
    Carrot
    Carrot is a Vegetable
    Apple
    Apple is a Fruit
